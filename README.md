Platypodes is a lightweight, standalone mod that adds Platypus to Minecraft!
## Features
#### Platypus
- Clone of the Axolotl
- Found in rivers
- 4 natural spawning, common variants
- 1 rare variant from breeding
- Can be picked up with a water bucket
- Can be bred with Yabbies
#### Bucket of Platypus
- Acquired by using a water bucket on a Platypus
- Easily transport Platypus
#### Platypus Spawn Egg
- Spawn a Platypus
#### Yabby
- Acquired from fishing, fisherman villagers and village chests
- Used to breed Platypus
